🌿 Smart Agri AI – Intelligent Farming Assistant

Smart Agri AI is an AI-powered web application designed to assist farmers in detecting plant diseases, analyzing crop conditions, and receiving smart agricultural guidance in both English and Tamil.

🚀 Features

🌱 Leaf Disease Detection
Upload or capture plant leaf images to detect diseases using a trained CNN model.

🔥 AI Heatmap Visualization
Highlights infected areas on the leaf using saliency maps.

🌦 Weather Advisory System
Provides real-time weather updates and temperature-based suggestions.

🌿 Fertilizer Recommendation
Suggests suitable fertilizers based on weather conditions.

📷 Camera Integration
Capture leaf images directly from your device camera.

🎤 Tamil Voice Assistant
Farmers can interact using Tamil voice commands.

🔊 Voice Output (English + Tamil)
AI provides spoken results in both languages.

🤖 Farmer AI Chatbot
Ask questions about crops, diseases, and fertilizers in Tamil or English.
